simplelikes widget

Drop-in like buttons for your static site. No build step needed.

Heart (default)

Thumbs-up

Star

Usage

<simple-likes slug="my-post"></simple-likes>

<simple-likes slug="my-post" data-icon="thumbs-up"></simple-likes>

<simple-likes slug="my-post" data-icon="star" class="custom"></simple-likes>

<script src="simple-likes.js"></script>
<script>
  window.__simpleLikesApiUrl = "https://simplelikes.william-brendaw.workers.dev";
</script>

Customization

<style>
  simple-likes {
    --sl-color: #9b59b6;        /* default: currentColor */
    --sl-color-active: #8e44ad;  /* default: #e74c3c */
    --sl-size: 24px;             /* default: 20px */
  }
</style>
